Git Quiz

What is Git and why is it so important?

Git is an online code repository that coders use to share their works and it is important for purposes of collaboration and backtracking any changes made on the code.

What's a branch and why do we create them?

A branch is a channel used to connect our local repository to the remote repository, branch changes are made to prevent us from fucking up a perfectly functional line or lines of code.

What's the working tree?

The working tree is the current files that we are working on, our current branch.

What's the staging area?

The staging area, also known as index, is where commits are prepared. It compares files in the working tree to the files in the remote repository.

What's the command to retrieve the latest version of your files from the remote server?

git pull
